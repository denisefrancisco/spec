spec

spec ([SP]ecial [E]vents [C]alendar) is an event management application built with a focus on calendaring, staff and equipment management. spec was created by students working for Information Technology Services at Wesleyan University. It is currently used by Instructional Media Services for supporting campus events.

spec's frontend uses jQuery, Backbone.js, Bootstrap and FullCalendar.

On the backend, spec uses Node.js, MongoDB and various npm modules.

For documentation, please see the wiki.
